## Workflow Philosophy

AFTER-HOURS is atmosphere-sensitive.

Do not optimize away:
- silence
- spacing
- ambiguity
- restraint
- slow pacing
- cinematic transitions

Preserve tactile feeling over feature density.

This is not a productivity dashboard.
This is not a startup landing page.
This is not a modern SaaS UI.

The experience should feel:
- late-night
- handcrafted
- slightly imperfect
- quiet
- spatial
- lived-in

Favor mood over feature count.
Favor consistency over novelty.
Preserve soul at all costs.
